# weather-20878

A simple static weather dashboard built with HTML, CSS, and JavaScript.

## What it does

- Shows current temperature, precipitation, humidity, and wind.
- Displays an hourly chart and 7-day forecast.
- Uses the free Open-Meteo APIs for location search and weather data.
- Works without requiring a GitHub account.

## Run locally

1. Open `index.html` in a browser, or use a local server.
2. Search for a city or ZIP code.
3. The dashboard updates automatically.

## Publish with GitHub Pages

- A workflow is included at `.github/workflows/pages.yml`.
- On every push to `main`, GitHub Pages will build and publish the site.
- The public URL will be:
  `https://deyongxu-rgb.github.io/weather-20878/`

## Files

- `index.html` — page structure
- `styles.css` — design and layout
- `script.js` — weather data fetch and rendering
